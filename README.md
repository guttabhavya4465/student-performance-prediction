# Student Performance Prediction

This project uses machine learning techniques to predict student performance based on different factors. The dataset contains student-related details like:

- Study Time  
- Previous Scores  
- Attendance  
- Parental Education  

The goal is to understand how these factors affect student performance and build a model to make predictions.

## Dataset  
Dataset used: Student Performance Dataset  

Features used:
- Study Time  
- Previous Scores  
- Attendance  
- Other relevant attributes  

## Technologies Used  
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

## Steps Performed  
- Imported dataset  
- Performed data preprocessing  
- Analyzed data using visualizations  
- Selected important features  
- Applied machine learning model  
- Evaluated model performance  

## Output  
- Predicted student performance  
- Visualization of relationships between features  
- Model evaluation results  

## How to Run  
- Open the notebook in Google Colab  
- Upload dataset  
- Run all cells  

## Conclusion  
Machine learning models can help in predicting student performance and identifying key factors that influence academic success.
